# YouTube Data Analysis Project

## Overview
Comprehensive analysis of YouTube trending videos and user engagement patterns across multiple countries.

## Features
- Sentiment analysis of user comments
- Word cloud generation
- Emoji usage analysis
- Video category performance analysis
- Channel trending analysis
- User engagement metrics
- Content characteristics analysis

## Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- TextBlob, WordCloud
- SQLite

## Results
• Average Sentiment Polarity: 0.138
• Most Engaging Category: Music
• Top Trending Channel: The Late Show with Stephen Colbert
• Average Like Rate: 3.64%
